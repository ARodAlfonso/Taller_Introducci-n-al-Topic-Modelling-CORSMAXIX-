En este repositorio se encuentran todos los archivos necesarios para el análisis a través de un modelado de tópicos del corpus de revistas españolas CORSMAXIX: un archivo con los textos del corpus lematizados, un documento txt. con una lista de "stop words" frecuentes en español, ampliada con vocablos específicos adaptados al corpus, así como una lista de comandos básicos en Mallet para explorar el corpus.
